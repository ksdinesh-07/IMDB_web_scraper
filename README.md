# IMDb Web Scraper

This project is a Python-based web scraper that collects data from IMDb's Top 250 movies list and saves it into a CSV file.

## Features

- Scrapes movie titles, ratings, and other relevant details from IMDb Top 250 movies page.
- Saves the scraped data into a CSV file (`imdb_top_250_movies.csv`).
- Simple and easy to use.

## Requirements

- Python 3.x
- `requests`
- `beautifulsoup4`
- `pandas`

Install dependencies with:

```bash
pip install -r requirements.txt
